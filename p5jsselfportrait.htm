P5.js Self Portrait

In this project, I practiced using P5.js which is a variant of the JavaScript coding language. I used the application Processing to code the piece. The piece itself is a translation of what I felt my face would look like in coding.

While it is super simple and does not contain great detail, I think it's cute and since it took me many hours to figure it out, I am proud of myself.

Here's my code!

function setup() {
  createCanvas(600, 600);
}
function draw() {
  background(204, 112, 92); 
  
   //draw hair 3
  fill(110, 75, 34); 
  ellipse(300, 320, 300, 400);
  
  //draw head
  fill(217, 178, 124); 
  ellipse(300, 300, 270, 295);
  
  //draw eye 1
  fill(250); 
  ellipse(240, 275, 80, 50);
  
  //draw eye 2
  fill(250); 
  ellipse(355, 275, 80, 50);
  
  //draw pupil 1
  fill(84, 59, 25);
  circle(250, 275, 40, 40);
  
  //draw pupil 2
  fill(84, 59, 25);
  circle(350, 275, 40, 40);
  
  //draw mouth
  noFill();
  stroke(1);
  arc(280, 370, 40, 20, 0, PI);
  arc(320, 370, 40, 20, 0, PI);
  
  //draw nose
  fill(201, 164, 113);
  triangle(300, 280, 325, 340, 277, 340);
  
  //hair 1
  fill(110, 75, 34);
  bezier(300, 153, 70, 100, 200, 400, 165, 350);
  
  //hair 2
  fill(110, 75, 34)
  bezier(300, 153, 530, 100, 400, 400, 435, 350);

}
