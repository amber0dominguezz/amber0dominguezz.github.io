DIY Photoshop

My DIY Photoshop piece is based on developing our skills using P5.js coding and is a continuation of the previous Self Portrait piece.

Particularly, this piece is based on, what can you create using your 10 number keys on a keyboard?

And so, I made my piece to be different versions of Hello Kitty that you can customize using different accessories I provided! Everything was hand drawn using Procreate, the iPad Air 5th Gen, and a 2nd Gen Apple Pencil.

1-4 different Hello Kittys, 5-0 accessories, X to delete all, P to save your work

Heres my code!

function clear_print() {
// this will do one of two things, x clears the screen by resetting the background
// p calls the routine saveme, which saves a copy of the screen
  if (key == 'x' || key == 'X') {
    background(img); 
  } else if (key == 'p' || key == 'P') {
     saveme();  
  }
}

function saveme(){
  filename=initials+day() + hour() + minute() +second();
  if (second()!=lastscreenshot) { 
    saveCanvas(filename, 'jpg');
    key="";
  }
  lastscreenshot=second();
  
}
